# Marcan WordPress Theme

Theme WordPress personalizado basado en el archivo Figma `NEW-MARCAN-2`.

## Qué incluye

- Intro estilo Figma con fondo amarillo, logotipo `marcan`, franja inferior gris y animación de entrada.
- Tipo de contenido administrable `Propiedades`.
- Taxonomías `Tipos de propiedad` y `Ubicaciones`.
- Campos nativos para precio, área, dormitorios, baños, estacionamientos, dirección y CTA.
- Plantillas PHP para home, archivo de propiedades, ficha de propiedad y páginas.

## Instalación rápida

1. Copia la carpeta `marcan-theme` dentro de `wp-content/themes/`.
2. En WordPress entra a `Apariencia > Temas`.
3. Activa el theme `Marcan`.
4. Entra a `Ajustes > Enlaces permanentes` y presiona `Guardar cambios`.
5. Entra a `Propiedades > Agregar propiedad` para cargar inmuebles.

## Administración de propiedades

Cada propiedad se administra desde el panel de WordPress:

- Título.
- Descripción.
- Imagen destacada.
- Extracto.
- Tipo de propiedad.
- Ubicación.
- Precio.
- Área.
- Dormitorios.
- Baños.
- Estacionamientos.
- Dirección.
- Botón de consulta.

## Pendiente para fidelidad completa

El Figma compartido apunta al frame `Intro B01`. Para dejar el sitio exactamente igual al diseño completo, hay que mapear los demás frames finales de la web y convertirlos a secciones o plantillas WordPress.
